# MECA0011-2-Fluid-Mechanics
Modélisation d'un écoulement irrotationnel
Projet de modélisation d'un écoulement irrationnel réalisé avec Matlab dans le cadre du cours d'Eléments de mécanique des fluides donné en
bachelier en sciences de l'ingénieur, orientation ingénieur civil (Université de Liège, 2018-2019).
